# kjnotebooklm

2026-06-28 한콘연 온라인 특강 **NotebookLM A to Z** 후기 선물 페이지 정적 파일입니다.

## Files

- `index.html` — 후기선물 안내 페이지 (요약본 + 강의 링크 + 복사해서 쓰는 프롬프트)
- `buyer.html` — 구매선물 안내 페이지 (요약본 + 강의 링크 + 프롬프트 + 구매자 전용 Canva 자료)
- `summary.html` — 6/28 강의 핵심 내용 요약본 약식 페이지
- `cover.png` — 표지 이미지 (후기 선물 안내 포스터, 1672×941)
- `meeting_chat.txt` — 강의 채팅 로그 원본 (페이지에는 사용 안 함, 참고용)

## Vercel
